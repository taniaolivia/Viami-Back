Supprimer mon compte

Nous sommes désolés de vous voir partir et nous espérons que vous avez apprécié notre application. Si vous souhaitez supprimer définitivement votre compte et toutes les données associées, veuillez suivre les étapes suivantes :

  • Ouvrez le menu de l'application en cliquant sur l'icône en haut à gauche de l'écran.
  • Cliquez sur "Paramètres".
  • Cliquez sur "Supprimer mon compte".
  • Confirmez votre demande en cliquant sur "Supprimer".

Quelles données seront supprimées ?

En supprimant votre compte, vous supprimerez définitivement toutes les données associées à votre compte, y compris vos discussions, votre historique de chat, vos publications sur le forum, vos commentaires, vos photos et vos informations personnelles telles que votre adresse e-mail, votre numéro de téléphone et votre localisation. Veuillez noter que cette action est irréversible et que vous ne pourrez pas récupérer vos données une fois votre compte supprimé.
